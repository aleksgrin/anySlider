## Примеры формирования объекта options
Для кругового слайдера необходимо задать только один параметр: радиус окружности
```js
const options = {
  type: {
    curve: 'circle',
    r: 100
  }
}
```
Для спирального слайдера необходимо задать: начальные и конечные углы и радиусы

```js
const options = {
  type: {
    curve: 'spiral',
    fi1: 0,
    fi2: 720,
    r1: 0,
    r2: 200
  }
}
```
```js
const options = {
  type: {
    curve: 'line',
    x1: 100,
    x2: 400,
    y1: 100,
    y2: 200
  }
}
```
